=== MASTER ARTIST PORTRAITS / AVATARS ===

Upload all master artist profile pictures directly to this folder:
/public/images/avatars/

To replace any artist profile photo, name your file using their unique lowercase ID:
[artist-id].[extension]

Examples:
- fede-almanzor.png
- gkirin.jpg
- jordan.png
- toni-garcia.jpeg

Supported formats: .png, .jpg, .jpeg, .JPG, .PNG, .JPEG

If you upload a file with one of these names and extension, the website will instantly find and use it. If there is no local file for an artist, the applet automatically falls back to curated portraits.

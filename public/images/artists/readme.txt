=== ARTIST IMAGES DIRECTORY ===

All artist-related images are placed directly in this folder. There is NO need to create subfolders or deep nested directories.

For any artist (using their exact lowercase ID, e.g., "jordan", "fede-almanzor", "toni-garcia"):

1. Profile Picture:
   Name it exactly: [artist-id].png
   Example: jordan.png

2. Portfolio Tattoo Images (up to 4 pieces per artist):
   Name them exactly: [artist-id]-portfolio-[number].png
   Examples:
   - jordan-portfolio-1.png
   - jordan-portfolio-2.png
   - jordan-portfolio-3.png
   - jordan-portfolio-4.png

Supported formats: .png, .jpg, .jpeg, .JPG, .PNG

If you upload an image, it will automatically load and replace the placeholder.
